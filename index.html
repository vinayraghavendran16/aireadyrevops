The framework

Six dimensions. One Index.

Each dimension contains four scoring criteria, rated 1 to 5 against a published rubric. Dimension scores are weighted into a single 0–100 Readiness Index that decomposes cleanly into the underlying drivers.

01 / Foundation

Data Foundation

Weight 20%

Field completeness, picklist discipline, deduplication, and data ownership. AI is only as good as the records it queries.

02 / Architecture

Integration Architecture

Weight 15%

System-of-record clarity, sync health, and data lineage. Without one truth, every AI agent inherits the conflict.

03 / Process

Process Standardization

Weight 15%

Lead-to-cash, stage exit criteria, MQL/SQL alignment, handoff SLAs. AI cannot infer process — it can only execute it.

04 / Stack Fit

AI Stack Fit

Weight 20%

Tool inventory, ROI tracking, adoption, data pipeline quality, use-case discipline. Where wasted spend hides.

05 / Governance

Governance & Access

Weight 15%

Connected apps, OAuth grants, profiles, audit trails. Post-breach, governance is a revenue concern, not just an IT one.

06 / Trust

Forecasting Trust

Weight 15%

Accuracy, pipeline hygiene, methodology, and whether leadership actually believes the number. The integration test for everything below.
